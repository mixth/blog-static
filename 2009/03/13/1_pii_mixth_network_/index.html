1 ปี Mixth Network!

Narat S

Mar 13, 2009

23 มีนานี้แล้วครับ จะครบรอบหนึ่งปีของ Mixth Network อย่างเป็นทางการ  สำหรับโปรเจ็ค Mixth Network นี้ทำอะไรมาแล้วบ้าง?

เรามี Aquarian ครับ เว็บไซด์ของเพื่อนๆ ที่รักแฟนฟิค แฟนอาร์ต แล้วก็นิยายครับ  Aquarian น่าจะตามมาติดๆกับ Mixth Network สิริอายุไม่น่าเกิน 6 เดือนครับ  เป็นเว็บแรกที่ผมมีการทำงานที่แปลกๆ คือเป็นแอดมินที่ไม่ได้สนใจเรื่องของเว็บไซด์เลย ในขณะที่ต้องทำงานกับมันตลอดเวลา

นอกจาก Aquarian แล้ว ก็มีเว็บบล็อกส่วนตัว Eternal-Terminal แห่งนี้ครับ  ส่วนมากแล้วก็บ่นเรื่องตัวเองให้ฟัง พอมีเวลาว่างก็หาอะไรมาให้เพื่อนๆได้ดูกัน Eternal-Terminal เกิดมาได้ 3 ปีแล้ว เมื่อก่อนอยู่ใน Spaces บริการโดย Microsoft ถัดมาก็ตั้งที่ Wordpress.com ก่อนจะกระโจนมารัน Wordpress บนโฮสต์ของตัวเองครับ... หลังจากนั้นก็ไม่ค่อยมีอะไรเป็นสาระ ก่อนหน้านี้ก็เว็บข่าวการเมืองภาษาอังกฤษ แล้วก็เว็บบ่นๆๆ ล้างกันหลายรอบ เปลี่ยนหน้ากากกันไปหลายครั้ง แต่ก็ขอบคุณทุกๆท่านที่ยังฟังผมบ่นอยู่นี้ ที่เข้ามาอ่านกันครับ

โปรเจ็ค RNM อีกโปรเจ็คที่กระโจนเป็น Public ไม่ถึงสัปดาห์นี้  ความจริงตั้งแต่จบเทอมหนึ่งก็มีการ public ไฟล์บ้างแล้ว แต่ครั้งนี้ทำใหม่ให้มี Template ที่สวยงาม ใช้งานสะดวกอีกครั้งครับ

ถัดมาก็ใหม่ล่าสุด กับ AFSer.in.th ถือว่าเป็นจุดที่ประสบความสำเร็จมากที่สุดเท่าที่ทำมาครับ แต่ก่อนมี @golfth มาช่วย หลังจากนั้นมี @snory เข้ามาช่วยในการประชาสัมพันธ์และแนะนำต่างๆ หลังจากจ่ายค่าโฮสต์ไปแล้ว ก็เหมือนจะทำอะไรได้เ้ต็มที คาดว่าส่วนนี้จะมีการเจริญเติบโตยิ่งๆขึ้นไปอีกครับ

ขอขอบคุณสำหรับการติดตาม Mixth Network ครับ คาดว่าในปีถัดไป จะมีผลงานที่สร้างประโยชน์ให้เพื่อนๆ มากที่สุดเท่าที่ทำได้  ซึ่งแน่นอนยังคงยึดหลักเดิมคือไม่มีโฆษณา ไม่มีผลประโยชน์ทางการค้า หากเป็นไปได้ก็ทำบนสัญญา CreativeCommons ชนิดนั้นก็แล้วแต่งานที่ทำครับ ^^
